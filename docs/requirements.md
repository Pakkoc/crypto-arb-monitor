[프로젝트 개요]

- 가상화폐 거래소 실시간 차익 모니터링 및 알림 발송

[프로젝트 배경 및 목표]

- 본 프로젝트는 국내외 주요 가상화폐 거래소 5곳(빗썸, 업비트, 바이낸스, 바이비트, 코인원)의 데이터를 연동하여, 각 거래소 간의 실시간 코인 가격 차이(스프레드)를 모니터링 후 텔레그램으로 알림을 전송하는 것을 목표로 합니다.

[과업 범위]

1. 수행 범위

- 프런트엔드/Client 개발
- 백엔드 개발

2. 상세 기능 요구 사항
   2-1. 거래소 API 연동: 빗썸, 업비트, 바이낸스, 바이비트, 코인원 총 5개 거래소의 실시간 시세 데이터 연동 및 수집
   2-2. 사용자 알림 설정: 사용자가 직접 원하는 차익 퍼센트(예: 3%)를 입력하고 설정할 수 있는 기능 제공
   2-5. 텔레그램 연동 알림 발송: 설정한 퍼센트 이상의 차익이 발생했을 때 텔레그램을 통해 알림 전송

[지원 디바이스]

- PC Web

[기술 스택]

- 프로젝트에 가장 적합하다고 판단되는 기술 스택을 자유롭게 제안해 주시기 바랍니다.

[지원 자격 및 우대 사항]

1. 지원 자격

- 가상화폐 거래소 API 연동 개발 경험

2. 우대 사항

- 코인 차익 거래(김치 프리미엄) 관련 모니터링 서비스 구축 경험자 우대
- 대용량 실시간 데이터 처리 및 대시보드 개발 경험자 우대

[산출물]

- 개발 소스 코드 원본
